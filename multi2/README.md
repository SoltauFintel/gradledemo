# Gradle Beispiel 6: Multi-Project-Build

Dies ist eine JAR-Bibliothek, die von multi genutzt wird. Bei der Entwicklung in Eclipse sind multi2-Änderungen in multi sofort sichtbar.
Beim Build wird multi2 mitgebaut und als Teil der multi Application releaset.

Die Eclipse-Dateien wurden nicht eingecheckt. Durch Ausführung von Target "eclipse" der Ant-Datei ../multi/build.xml werden die Eclipse-Dateien erstellt.
